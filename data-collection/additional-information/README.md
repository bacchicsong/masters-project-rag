В данной директории расположена информация о сборе дополнительных данных для базы знаний, представляющих собой обзор основных тем, связанных с **российским фондовым рынком** и полезных частному инвестору.

## Источник данных

Данные берутся с сайта:  
[https://www.tbank.ru/invest/help/educate/how-it-works/](https://www.tbank.ru/invest/help/educate/how-it-works/)

## Процесс сбора

### Алгоритм работы:
- Скрапер переходит по ссылкам на изначальной странице
- Собирает информацию с новых страниц
- Рекурсивно переходит по ссылкам уже на новых страницах

### Настройки:
- **Глубина перехода**
- **Задержка между запросами**
- **Таймаут запроса**

### Сохранение данных:
Итоговая информация хранится в json файлах. Наполнение каждого файла отражает тему каждой отдельной страницы.
**Структура json**:
```json
{
  "url": "https://example.com/page",
  "title": "Заголовок статьи",
  "content": "Основной текст статьи...",
  "sections": [
    {
      "heading": "Заголовок раздела",
      "content": ["Текст раздела..."]
    }
  ]
}
```
