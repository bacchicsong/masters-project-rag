## Сбор данных по ценам и основным характеристикам финансовых инструментов и их эмитентов

Так как на данный момент доступа к API Cbonds нет (но есть подписка CBonds Professional), можно выгружать данные по ценам акций из открытого источника **investing.com**. В перспективе возможно либо начать парсить CBonds так же, но для доступа к ценам облигаций.

**investing_com_scraping_prices.py** - код, который парсит с сайта Investing.com данные по ценам всех доступных компаний, торгуемых на Московской Бирже.


**investing_com_scraping_emitents.py** - код, который парсит с сайта Investing.com данные по характеристикам эмитентов:

1. Основная информация: Название компании, символ
2. Рыночные данные: Текущие цены, изменения, объем продаж
3. Профиль компании: Описание, сектор, отрасль промышленности
4. Финансовые показатели: Соотношение прибыли и убытка, финансовые коэффициенты
5. Технические индикаторы: RSI, скользящие средние
6. Финансовая отчетность: Доступность и ссылки
7. Информация о дивидендах: История дивидендов и доходность
