## Теоретические работы: учебники, книги, статьи

В данной папке расположена коллекция универсальных работ в сфере финансов и количественных финансов, которые будут использоваться LLM для теоретического обоснования рекомендаций, а также будет покрывать те кейсы, когда информации в данных из папки *"additional-information** не будет хватать. Например, когда клиенту будет инетерсно исследовать то, от каких величин зависит его портфель. 

### 1. Тексты методологии Московской Биржи

В текущей рыночной ситуации (осень 2025 года) наиболее доходными конвенциональными активами являются облигации, в связи с чем у клиента может возникнуть запрос на теоретическую оценку доходлности этих инструментов. Более продвинутым (квалифицированным) инвесторам может быть инетересно узнать методологию расчетов цен опционов и фьючерсов. В рамках работы предполагается, что клиенты торгуют инструментами, представленными на Московской Бирже

### 2. Книги по финансам

Клиент может узнать теоретически обоснованные методики аналика рынка для составления собственного портфолио из книги **Online Portfolio Selection Principles and Algorithms** и разобраться в сложных финансовых инструментах в книге Hull. Более простые вопросы современные LLM могут покрывать самостоятельно

## Парсинг текстов

С помощью скрипта parsing_texts.py данные из учебников (.pdf, .docx) агрегируются в json-формат и в перспективе будут положены в векторную базу данных (Milvus) для создания алгоритма быстрого поиска (B25) по ним.

Формат для .pdf файлов

```python 
{
  "file_name": "filename.pdf",
  "file_type": "PDF",
  "metadata": {...},
  "page_count": 100,
  "content": [
    {"page": 1, "text": "текст страницы..."}
  ]
}
```

Формат для .docx файлов:

```python 
{
  "file_name": "filename.docx",
  "file_type": "DOCX",
  "metadata": {...},
  "paragraphs": [...],
  "tables": [...]
}
```

`parsed_files.json` - пример распаршенного файла из нескольких представленных книг/методичек
