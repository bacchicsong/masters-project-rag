## Сбор данных по ценам и основным характеристикам финансовых инструментов и их эмитентов

Так как на данный момент доступа к API Cbonds нет (но есть подписка CBonds Professional), можно выгружать данные по ценам акций из открытого источника **investing.com**. В перспективе возможно либо начать парсить CBonds так же, но для доступа к ценам облигаций.
